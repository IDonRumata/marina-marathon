    git init
    git add .
    git commit -m "Initial commit"
    git remote add origin https://github.com/USERNAME/marina-marathon.git

2️⃣  VERCEL (3 минуты)

    • Add New → Project
    • Import marina-marathon

      - TELEGRAM_CHAT_ID = [ваш ID]
    • Deploy

    • Откройте ваш сайт
    • Заполните форму
    • Проверьте Telegram
